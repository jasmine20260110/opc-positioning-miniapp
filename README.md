# OPC定位神器（MVP Demo）

一个用于内部方向定位演示的微信小程序：用户完成 20 个问题后，系统结合个人证据、市场机会和现实约束，生成 3 条候选路线、1 条优先路线与 7 天最小行动计划。

> 当前为内部 MVP，不面向公众发布，不构成职业、投资或人生决策依据。

## 当前状态

- 阶段四：证据提取、路线生成、市场分析——已完成

## MVP流程

1. 首页继续定位，或使用示例数据快速演示；
2. 完成 20 个问题；
3. 云函数调用阿里百炼 `deepseek-v4-flash`；
4. 查看证据、3 条候选路线和市场机会；
5. 查看启动适配结论并选择路线；
6. 生成 7 天行动计划；
7. 点击“开始 Day 1”，在本机记录当前计划状态。

AI 输出会经过 JSON 解析和 Schema 校验。AI 调用失败时，前端允许重试，并为 7 天计划提供固定模板降级。

## 技术结构

```text

  pages/                  首页、问答、分析、报告、计划页面
  utils/                  数据契约、本地状态、规则和降级计划
cloudfunctions/opcApi/    AI 云函数、提示词、校验器和单元测试
scripts/                  本地、云端及开发者工具回归脚本

```

## 本地运行

1. 安装并打开微信开发者工具。
2. 导入本仓库根目录，确认 AppID 为自己的测试小程序 AppID。
3. 在 `miniprogram/app.js` 的 `globalData.env` 中填写微信云开发环境 ID；也可以在开发者工具中选择对应云环境。
4. 编译小程序。

成功标志：模拟器能显示“OPC定位神器”首页，点击“继续定位”可进入问答页。

## 配置AI云函数

云函数位于 `cloudfunctions/opcApi`。真实 API Key 只能配置在微信云开发控制台的云函数环境变量中，不能写入小程序代码、README 或 `.env.example`。

| 环境变量 | 必填 | 说明 |
|---|---|---|
| `DASHSCOPE_API_KEY` | 是 | 阿里百炼 API Key |
| `DASHSCOPE_BASE_URL` | 推荐 | 百炼业务空间的 OpenAI 兼容地址 |
| `DASHSCOPE_MODEL` | 否 | 默认 `deepseek-v4-flash` |
| `DASHSCOPE_TIMEOUT_MS` | 否 | 默认 `25000` 毫秒 |
| `DASHSCOPE_MAX_COMPLETION_TOKENS` | 否 | 默认 `6000` |

配置步骤：

1. 在微信开发者工具中选择当前云环境；
2. 上传并部署 `opcApi`，选择云端安装依赖；
3. 在云函数配置中开启公网访问；
4. 把执行超时设为 60 秒；
5. 添加上述环境变量并保存，然后重新部署或重启实例。

成功标志：调用云函数 `health` 操作返回 `configured: true`，且返回内容不包含 API Key。

更完整的小白配置说明见 [docs/阶段四-阿里百炼配置与验证.md](docs/阶段四-阿里百炼配置与验证.md)。

## 数据与隐私

- 用户答案会经微信云函数发送到阿里百炼进行分析；
- 云函数不主动记录完整答案或模型原文；
- 问答进度、分析结果和计划保存在小程序本机，不写入云数据库；
- 内部测试也不要输入身份证号、客户名单、公司机密等敏感内容；
- `.env` 和 `.env.*` 默认被 Git 忽略，`.env.example` 只能保存变量名和占位符。

## 当前限制

- 仅完成 P0 内部 Demo，没有登录、跨设备同步、云端历史记录和运营后台；
- Day 1 只记录本地开始状态，完整执行和提醒功能尚未实现；
- AI 结果用于方向验证参考，不能替代用户自己的判断；
- 微信体验版仍需完成 iPhone、安卓和非开发者账号人工验收。
